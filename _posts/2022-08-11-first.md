```
Stack 스택

 1. 스택이란?
   ▸쌓아 올린다는 것을 의미
   ▸데이터를 차곡차곡 쌓아 올리는 형태의 자료구조
   ▸메모리 안 데이터들을 더욱 효율적으로 다루기 위해 만들어진 데이터 참조 방식
   ▸가장 마지막에 들어온 자료가 먼저 나가는 구조이다.
    LIFO(Last In First Out)
   ▸스택을 사용하는 예시 : 웹 브라우저 방문기록, 작업 실행 취소, 뒤로가기
 
 2. 스택 함수
   ▸pop() : 가장 마지막에 push된 데이터 반환 후 삭제
   ▸push() : 데이터 추가
   ▸peek() : 가장 마지막에 push된 데이터 확인
   ▸remove() : 가장 마지막에 추가된 데이터 제거
 
 3. 스택을 이용한 코딩테스트 문제 예시
   ▸백준 9012번 괄호(입력 괄호 문자열이 올바른 괄호 문자열인지 YES 또는 NO 출력)
    → 스택에 문자열의 문자 하나가 '('이면 push하고, ')'이면 pop해서 조건에 따라 스택이 비어있으면 YES 아니면 NO 출력
      (문자열에 ')' 우괄호가 더 많을 경우 스택이 비어지게 되므로 NO 출력)
   ▸백준 9093번 단어 뒤집기(문장이 주어졌을 때 단어를 모두 뒤집어서 출력)
    → 문자열의 문자를 하나씩 스택에 push하고 빈칸(" ")이 있는 경우 pop해서 뒤집어진 단어 모두 출력 
   
Queue 큐

 1. 큐란?
   ▸양쪽 끝에서 데이터의 삽입과 삭제가 각각 이루어짐
   ▸가장 먼저 들어온 자료가 먼저 나가는 구조이다.
    FIFO(First In First Out)
   ▸스택과 반대되는 개념
 
 2. 큐 함수
   ▸poll() : 첫번째 값을 반환하고 제거, 비어있으면 null
   ▸peek() : 첫번째 값 참조/확인
   ▸add() 또는 offer() : 데이터 추가
   ▸remove() : 첫번째 데이터 제거
   
```
